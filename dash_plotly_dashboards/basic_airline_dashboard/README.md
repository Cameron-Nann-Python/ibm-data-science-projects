# Basic Airline Dashboard

## Overview
The objective of this dashboard is view proportions of people traveling based on flight distance. The dashboard features a pie chart of this relation and is a simple implementation of a Dash app.

## Required Technologies
- Python 3.13
  
## Required Libraries
- Pandas
- Dash
- Plotly

## Files Included
- `dash_intro.py`: Python file containing app and dataset provided by IBM.
