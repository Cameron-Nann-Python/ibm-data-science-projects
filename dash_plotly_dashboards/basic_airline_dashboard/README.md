# Basic Airline Dashboard

![Airline Dashboard Desktop](https://github.com/Cameron-Nann-Python/ibm-data-science-projects/blob/main/dash_plotly_dashboards/basic_airline_dashboard/screenshots/dash_intro_desktop.png)

## Overview
The objective of this dashboard is view proportions of people traveling based on flight distance. The dashboard features a pie chart of this relation and is a simple implementation of a Dash app.

## Required Technologies
- Python 3.13
  
## Import Required Libraries
Install required libraries with the following command:
```
pip install -r requirements.txt
```

## Run the App
Open the dashboard on a local server with the following command:
```
python app.py
```

## Files Included
- `dash_intro.py`: Python file containing Dash app and IBM dataset.
- `requirements.txt`: Required Python libraries
- `dash_intro_desktop.png`: Dash app on desktop
